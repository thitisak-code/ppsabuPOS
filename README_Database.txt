🎉 เวอร์ชันใหม่! ใช้ SQLite Database แทน JSON
✅ เร็วกว่า ปลอดภัยกว่า ทนทานกว่า

📦 วิธีแปลงเป็นไฟล์ .exe:
─────────────────────────────────────────────────────────
1. เปิด Command Prompt
2. พิมพ์: python build_exe_db.py
3. กด Enter และรอสักครู่
4. ไฟล์ .exe จะอยู่ในโฟลเดอร์ "dist"
5. คัดลอกไฟล์ .exe ไปใช้ได้เลย!

🌟 ข้อดีของ Database Version:
─────────────────────────────────────────────────────────
✓ ค้นหาข้อมูลเร็วกว่า 10 เท่า
✓ รองรับข้อมูลนับหมื่นรายการ
✓ ปลอดภัยกว่า ข้อมูลไม่เสียหายง่าย
✓ บันทึกอัตโนมัติทันที (ไม่ต้องกดบันทึก)
✓ มีโครงสร้างข้อมูลที่ชัดเจน
✓ กู้คืนข้อมูลง่าย

💡 วิธีใช้งานโปรแกรม:
─────────────────────────────────────────────────────────
✓ เปิดโปรแกรม = ดับเบิลคลิก .exe
✓ ปิดโปรแกรม = คลิกปุ่ม "❌ ปิดโปรแกรม"
✓ ข้อมูลบันทึกอัตโนมัติ (ไม่ต้องกด)
✓ ค้นหาประวัติ = คลิก "📜 ประวัติการขาย"

📁 ไฟล์สำคัญ:
─────────────────────────────────────────────────────────
• main_with_database.py = โค้ดหลัก (ใช้ Database)
• database.py = โมดูลจัดการฐานข้อมูล
• build_exe_db.py = สคริปต์แปลงเป็น .exe
• shabu_pos.db = ไฟล์ฐานข้อมูล (สร้างอัตโนมัติ)
• คู่มือ_Database.md = คู่มือฉบับเต็ม

📊 โครงสร้างฐานข้อมูล:
─────────────────────────────────────────────────────────
• menu_items = รายการเมนูอาหาร
• tables = ข้อมูลโต๊ะ
• orders = ออเดอร์ปัจจุบัน (บิลที่ยังไม่ชำระ)
• sales_history = ประวัติการขาย
• sale_items = รายการในแต่ละบิล

⚠️ คำเตือน:
─────────────────────────────────────────────────────────
• อย่าลบไฟล์ shabu_pos.db (ข้อมูลจะหาย!)
• ควรสำรองข้อมูลเป็นประจำ
• แนะนำให้ปิดโปรแกรมเมื่อไม่ใช้

🔧 เครื่องมือเสริม (แนะนำ):
─────────────────────────────────────────────────────────
DB Browser for SQLite (ฟรี)
→ ดาวน์โหลด: https://sqlitebrowser.org/
→ ใช้ดู/แก้ไข/Export ข้อมูล
→ สร้างรายงานขั้นสูง

💾 วิธีสำรองข้อมูล:
─────────────────────────────────────────────────────────
1. คัดลอกไฟล์ shabu_pos.db
2. เปลี่ยนชื่อเป็น shabu_pos_backup_วันที่.db
3. เก็บไว้ในที่ปลอดภัย

🎯 เปรียบเทียบ JSON vs Database:
─────────────────────────────────────────────────────────
                   JSON        Database
ความเร็ว            ช้า         เร็วมาก ⚡
ข้อมูลสูงสุด        1,000       ไม่จำกัด 📊
ความปลอดภัย        ปานกลาง     สูง 🔒
การค้นหา           ช้า         ทันที 🔍
การกู้คืน          ยาก         ง่าย ✅

• ต้องการวิเคราะห์ยอดขาย
• เก็บข้อมูลประวัติยาวนาน
